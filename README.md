# i-fudget
Create File app.config.ts in src/app/ with content
```
export const firebaseConfig = {
  apiKey: "###################",
  authDomain: "###################",
  databaseURL: "###################",
  projectId: "###################",
  storageBucket: "##############",
  messagingSenderId: "########"
};
```

Replace values with your firebase configuration

Run npm install

Run ionic serve
