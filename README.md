# i-fudget
Create a file with the name `app.config.ts` in `src/app/` and paste below code :
```
export const firebaseConfig = {
  apiKey: "###################",
  authDomain: "###################",
  databaseURL: "###################",
  projectId: "###################",
  storageBucket: "##############",
  messagingSenderId: "########"
};
```

Replace values with your firebase configuration

Run `npm install`

Run `ionic serve`
